=== rclone --help (commands) ===
Available commands:
  completion  Generate the autocompletion script for the specified shell
  help        Show help for rclone commands, flags and backends.
  ls          List the objects in the path with size and path.
  mount       Mount the remote as file system on a mountpoint.
  mount2      Mount the remote as file system on a mountpoint.
  multimount  Mount multiple filestore filesystems simultaneously

Use "rclone [command] --help" for more information about a command.

=== help backend rclone-filestore ===
### Standard options

Here are the Standard options specific to rclone-filestore (Filestore Backend).

#### --rclone-filestore-url

Filestore service URL (handles file metadata + GCS operations internally)

Properties:

- Config:      url
- Env Var:     RCLONE_RCLONE_FILESTORE_URL
- Type:        string
- Default:     "http://localhost:9112"

#### --rclone-filestore-filesystem-id

Filesystem ID to use

Properties:

- Config:      filesystem_id
- Env Var:     RCLONE_RCLONE_FILESTORE_FILESYSTEM_ID
- Type:        string
- Required:    true

#### --rclone-filestore-auth-token

Authentication token (JWT)

**NB** Input to this must be obscured - see [rclone obscure](/commands/rclone_obscure/).

Properties:

- Config:      auth_token
- Env Var:     RCLONE_RCLONE_FILESTORE_AUTH_TOKEN
- Type:        string
- Required:    false

### Advanced options

Here are the Advanced options specific to rclone-filestore (Filestore Backend).

#### --rclone-filestore-description

Description of the remote.

Properties:

- Config:      description
- Env Var:     RCLONE_RCLONE_FILESTORE_DESCRIPTION
- Type:        string
- Required:    false
