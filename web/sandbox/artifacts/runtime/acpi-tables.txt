## Interpretation
  - Hardware-reduced ACPI (FADT bit20=1): no legacy 8259 PIC, no SMI, no PM1 blocks; events via GED.
  - _SB.GED_ (ACPI0013): Generic Event Device = the ACPI:Ged IRQs 24/25 in /proc/interrupts.
  - _SB.VGEN (HID FCVMGID, CID VM_Gen_Counter): VM Generation Counter — guest detects snapshot
    restore via this; explains 'crng reseeded due to virtual machine fork' (see doc 03 snapstart).
  - _SB.VCLK (HID AMZNC10C, VMCLOCK): Amazon paravirtual clock => Amazon/AWS Firecracker build.
  - _SB.PC00 + PHPR/PCEJ/_EJ0, slots S000-S007: PCI host bridge with 8 hotplug slots; ECAM 0xeec00000.
